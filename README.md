#Image Viewer JavaFX
##Requirements
```
JDK 1.8.0_311 (JDK contains javafx libraries)
Maven
```
##How to run
```
mvn compile assembly:assembly
cd ./target
java -jar App-jar-with-dependencies.jar
```
##App configuration without compiling
```
application.properties
```
##Log4j2 configuration
```
Log4j2.xml
```
